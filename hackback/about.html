About HackBack

The Podcast

HackBack is an offensive security podcast that discusses security highlights and insights. Each week we talk about the highlights for cybersecurity as well as exploring interesting security topics.

The idea of HackBack is to have a podcast that discusses various security topics while focusing on the technical insights.

HackBack is delivered in both English and Arabic.

The Host
